# New system Arrhenius

## Background 

End-of-life of Alvis, Tetralith and Dardel

- Alvis (GPU and AI) Summer 2026
- Tetralith Autumn 2026
- Dardel dec 2026-27

## What is Arrhenius 

- This is the new HPC cluster by NAISS.
- Vast increase of GPU performance
- Reduced CPU performance
- Reduced amount of nodes available for CPU calculations
- Lower RAM/CPU ratio
- Decrease in storage facility


- 2027: Just Arrhenius on national level up.

## What will happen?

- Active projects running on Dardel will be migrated to the new system

- Local clusters, like Kebnekaise, Pelle and Cosmos local university users several more years.
